# Omáda Téssera - Fashion & Accessories E-Commerce Micro Site

## Deskripsi Proyek

Omáda Téssera adalah situs mikro e-commerce sederhana untuk fashion dan aksesori. Proyek ini dibuat sebagai simulasi toko online dengan fitur-fitur dasar seperti katalog produk, keranjang belanja, checkout, dan manajemen akun. Dibangun menggunakan HTML, CSS, dan JavaScript murni tanpa framework eksternal.

## Fitur Utama

- **Navigasi Halaman**: Menu navigasi untuk berpindah antar halaman (Home, Catalog, Cart, Account, About).
- **Katalog Produk**: Tampilan produk dengan filter berdasarkan kategori dan pencarian.
- **Detail Produk**: Halaman detail produk dengan gambar, deskripsi, dan opsi ukuran/jumlah.
- **Keranjang Belanja**: Tambah, edit, dan hapus item dari keranjang.
- **Checkout**: Proses checkout dengan simulasi pembayaran dan pengiriman.
- **Manajemen Akun**: Simulasi login dan registrasi akun.
- **Responsif**: Desain yang dapat menyesuaikan dengan berbagai ukuran layar.

## Teknologi yang Digunakan

- **HTML5**: Struktur halaman web.
- **CSS3**: Styling dan layout responsif.
- **JavaScript (ES6+)**: Logika interaktif, manipulasi DOM, dan simulasi backend.

## Struktur File

```
/
├── coba.html          # File HTML utama dengan struktur dan styling
├── script.js           # JavaScript untuk fungsi interaktif
├── img/                # Folder gambar produk (tidak termasuk dalam repo ini)
│   ├── Male/
│   ├── Female/
│   ├── Accessories/
│   └── Religion/
└── README.md           # Dokumentasi proyek (file ini)
```

## Cara Menjalankan

1. **Clone atau Download**: Unduh semua file proyek ke komputer Anda.
2. **Buka di Browser**: Buka file `coba.html` di browser web modern (Chrome, Firefox, Safari, dll.).
3. **Navigasi**: Gunakan menu navigasi untuk menjelajahi fitur-fitur situs.

**Catatan**: Pastikan folder `img/` berisi gambar produk sesuai dengan path yang direferensikan dalam kode. Jika gambar tidak ada, produk akan ditampilkan tanpa gambar.

## Fitur Simulasi

- **Pembayaran**: Simulasi pembayaran dengan opsi transfer bank, kartu kredit, dan e-wallet.
- **Pengiriman**: Simulasi ongkir dengan pilihan kurir reguler atau express.
- **Akun**: Simulasi login/registrasi tanpa backend nyata.
- **Analytics**: Simulasi laporan penjualan dan pengunjung di halaman About.
